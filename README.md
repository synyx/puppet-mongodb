#mongodb

##Overview

A Puppet module to install and manage mongodb from 10gen repository.

Supported plattform:
* Debian 7
* Ubuntu 12.04
* CentOS 6.5

##Usage

All options and configuration can be done through interacting with the parameters
on the main mongodb class.


```puppet
class { 'mongodb':}
```
